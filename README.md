# Job Portal

A full-stack Job Portal web application built using Django.

## Features

- User Registration & Login
- Recruiter Dashboard
- Candidate Dashboard
- Add Jobs
- Apply for Jobs
- Save Jobs
- Company Listing
- Company Details
- Profile Management
- Application Status
- Search Jobs
- Bootstrap Responsive UI

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

## Installation

```bash
git clone https://github.com/ChristianEnan/Job-Portal.git
cd Job-Portal
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
